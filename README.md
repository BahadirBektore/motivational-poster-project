# Motivational Poster Project

The **Motivational Poster Project** is a simple web page that displays a motivational poster with a clean and visually appealing design.  
This project demonstrates **HTML and CSS skills**, focusing on layout, typography, and responsive design principles.

---

## 🌐 Live Demo

Check it out here:  
👉 https://bahadirbektore.github.io/motivational-poster-project/

---

## 🚀 Project Overview

This project includes:

- A visually appealing motivational poster  
- Attention to typography and layout  
- Fully responsive design suitable for different screen sizes  
- Clean and semantic HTML structure

---

## 🛠 Technologies Used

This project was built using:

- **HTML5** – Semantic structure  
- **CSS3** – Styling and layout, including responsive design
